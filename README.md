# hand-gesture-control system
## Project Description 
Hand gesture control system using python ,openCV ,and Mediapipe
This project allows users to control mouse,Powerpoint slides, and system functions using
hand gestures using computer vision 

## Features
1.mouse control using index finger
2. select  using pinch gesture
3.powerpoint slide control

##TECHNOLOGIES USED
. python
. openCV
. Mediapipe
. PyautoGUI
